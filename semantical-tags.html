Etiqueta main: Etiqueta que engloba el contenido principal de la pagina actual.

Etiqueta section: Es utilizada para englobar una o mas etiquetas con contenido relacionado entre si

Articulo:

Como su nombre lo indica, es la etiqueta que engloba un conjunto de etiquetas y/o contenido relacionado con un articulo

Etiqueta aside: Se utiliza para colocar contenido relacionado al contenido principal sin que este sea dependiente de la misma.

Etiqueta footer: Todo aquel contenido que suele aparecer al final de una pagina web (datos de contacto, otro menu, redes sociales, mapa, etc)
